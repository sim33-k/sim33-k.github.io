Project Description

An advanced logistics optimization system designed to solve complex vehicle routing problems for tea leaf collection operations. This system extends the Evergreen platform with intelligent route planning capabilities.

Key Features

  • CVRP Solver: Solves Capacitated Vehicle Routing Problems using constraint programming
  • Load Optimization: Minimizes truck under-loading and over-loading scenarios
  • Route Planning: Generates optimal collection routes considering time windows
  • Distance Calculation: Real-time distance matrix using Google Maps API
  • Constraint Management: Handles multiple constraints including vehicle capacity, time windows, and driver availability
  • Real-time Adjustments: Dynamic route recalculation based on actual conditions

Technical Highlights

The system leverages OptaPlanner, a powerful constraint satisfaction solver, to optimize vehicle routes. It uses various optimization algorithms including Tabu Search and Simulated Annealing to find near-optimal solutions for complex routing problems. The backend is built with Spring Boot for robustness and scalability.

Problem Solving Approach

  • Implemented custom score calculators for domain-specific constraints
  • Designed constraint streams for efficient problem evaluation
  • Integrated Google Maps Distance Matrix API for accurate travel time estimation
  • Optimized solver configuration for fast solution generation
  • Built RESTful APIs for integration with the Evergreen platform

Impact

This optimization system significantly reduced fuel costs and collection time for tea factories by providing intelligent route planning. It minimized vehicle under-utilization and ensured timely collection from all suppliers.
