Project Description

CarLog is a comprehensive mobile application designed to help vehicle owners track and manage all aspects of their vehicle maintenance, expenses, and diagnostics in one place.

Key Features

  • Trip Tracking: Log and analyze your driving trips
  • Mileage Logging: Automatic and manual mileage recording
  • Fuel Expense Tracking: Monitor fuel consumption and costs
  • Maintenance History: Complete service record management
  • OBD2 Integration: Real-time vehicle diagnostics via OBD2 adapter
  • Service Reminders: Automated maintenance notifications
  • Expense Reports: Detailed analytics and visualizations

Technical Highlights

The application uses React Native for cross-platform mobile development, ensuring consistent user experience on both iOS and Android. The backend is built with Java Spring Boot, providing robust RESTful APIs. Local data storage with SQLite ensures offline functionality, while PostgreSQL manages server-side data.
