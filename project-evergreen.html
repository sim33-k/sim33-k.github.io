Project Description

Evergreen is a comprehensive web platform designed for low-country tea factories to manage their entire supply chain operations. Built entirely from scratch using custom MVC architecture without relying on external PHP frameworks.

Key Features

  • Collection Scheduling: Automated scheduling for tea leaf collection from suppliers
  • Route Planning: Integrated Google Maps for optimal collection routes
  • Inventory Management: Real-time tracking of tea leaf stock and quality grades
  • Payment Tracking: Automated payment calculation based on weight and quality
  • Analytics Dashboard: Comprehensive reports and data visualization
  • Multi-user System: Role-based access for factories, suppliers, and collectors
  • Invoice Generation: Automated invoicing and payment records

Technical Highlights

The entire application was built from the ground up using pure PHP with a custom MVC architecture. This approach provided deep understanding of web application fundamentals and allowed for complete control over the system's behavior. The application is deployed on Google Cloud Platform for reliability and scalability.

Architecture & Design

  • Implemented custom MVC pattern without frameworks
  • Built custom routing system for clean URLs
  • Designed database schema for complex supply chain relationships
  • Created reusable template system for views
  • Implemented secure authentication and authorization
  • Built custom ORM-like data access layer

My Contributions

  • Designed and implemented the database schema
  • Developed the collection scheduling module
  • Integrated Google Maps API for route planning
  • Built the analytics dashboard and reporting system
  • Implemented the payment calculation engine

Impact

The platform improved operational efficiency for tea factories by automating manual processes, reducing paperwork, and providing real-time visibility into the supply chain. It helped ensure fair payment to suppliers and optimized collection routes.
