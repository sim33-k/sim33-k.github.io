Project Description

A modern and intuitive music player UI/UX design created in Figma, featuring a sleek interface for managing and enjoying music collections. The design focuses on user experience with easy navigation between songs, playlists, and albums.

Design Components

  • Now Playing screen with album artwork and controls
  • Library view with songs, playlists, and albums tabs
  • Search functionality with filters
  • Playlist management interface
  • Album detail pages with track listings
